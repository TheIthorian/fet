# Template Service

## Getting Started

Run the development server:

```bash
npm ci
npm run dev
```

Run production server:

```bash
npm run build
npm start
```

Uses default port of `301x`

## Api Endpoints

| Route                          | Description                            |
| ------------------------------ | -------------------------------------- |
| `GET api/health`               | Check service health                   |
| `GET api/location-information` | Get details about the current location |
